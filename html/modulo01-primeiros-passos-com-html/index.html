Aula 3: Estrutura Básica do HTML

Tá de marcação comigo? :)

A estrutura basica do HTML é composto pelas tags de tipo de documento doctype. O html, que define a estrutura do documento.
O head define o cabecalho do documento.
E o body define o corpo do documento.

Aula 4: Falando Sobre Tags

Compondo o HTML

As tags html ajudam a criar o documento, seguindo o(s) padrão(ões) aceito(s).
Temos tags específicas para texto. Tem tags para imagens, links, etc.
Ainda tem aquelas usadas para capturar informações fornecidas pelo usuário, como as usadas nos formularios.

Aula 6: Textos

Escrevendo...

Alguns tipos de sites são simplesmente textos. Ou o foco principal do seu negócio é textos. Por exemplo, sites de notícias, blogs, etc.
Temos tags para textos, como p para paragrafos, h1-h6 para cabeçalhos, exemplo:

    texto h1

    texto h2

    texto h3

    texto h4

    texto h5
    texto h6
bloquotes para citações, etc.

Descrevendo coisas

As vezes queremos mostrar uma lista de alguma coisa qualquer para o usuário do nosso site.
